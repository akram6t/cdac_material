1System Overview

AuctXI is a comprehensive web-based auction management platform designed to digitize and streamline player auctions for team-based sports leagues (T20 Cricket, etc.). The platform covers the entire auction lifecycle — from player pool creation and rule configuration to real-time bidding, payment processing, and team roster finalization.

Built around a role-based architecture, AuctXI serves three distinct user types: System Administrators who govern the platform, Auction Managers who orchestrate auction events, and Team Owners who participate as bidders. Each role has a dedicated dashboard and workflow tailored to their responsibilities.

2Problem Statement & Objectives

Problem: Traditional sports league auctions are conducted manually or through fragmented tools, leading to inefficiencies in bid tracking, lack of transparency, errors in budget management, and poor record-keeping. There is no unified platform that handles player pools, real-time bidding, rule enforcement, and payment processing in a single system.

Project Tagline
"Bid. Build. Win your team."

Objectives:

  • ✓Provide a single unified platform for end-to-end auction management
  • ✓Enable real-time competitive bidding with automatic rule validation
  • ✓Enforce configurable auction rules (base price, bid increment, purse limits, squad size)
  • ✓Deliver role-based dashboards with relevant analytics for each actor
  • ✓Track complete bid history and maintain transparent audit trails
  • ✓Integrate payment processing for seamless post-auction settlements
  • ✓Support multiple concurrent auctions with independent configurations
  • ✓Generate comprehensive reports for administrators and managers

Scope: The system is designed for T20 cricket league auctions but is architected to be adaptable for any team-based sport. The current implementation supports features such as player watchlists, live auction timers (3-minute bid windows), auction timelines with phase tracking, team purse management, and top-player analytics.

6System Flow

The primary end-to-end auction flow within AuctXI:

1
Admin Configures Rules — Sets auction_rules (base price, bid increment, purse limit, squad size, bid timer).
2
Manager Creates Auction — Registers event: name (e.g., "Royal Strikers T20"), schedule, venue, linked rule set.
3
Registration Phase Opens — Timeline begins; teams can register and join the auction (deadline tracked).
4
Manager Manages Player Pool — Populates players with profiles: name, role (Batsman/Bowler), nationality, stats, base price.
5
Team Owners Login & Prepare — Each owner authenticates, views player pool, builds watchlist, checks purse budget.
6
Manager Triggers Bidding Round — <<trigger>> presents a player; 3-minute countdown starts.
7
Team Owners Place Bids — Real-time competitive bidding; system validates increment rules and remaining purse.
8
Bid Closes → Player Assigned — Timer expires or no higher bid; team_player record created; purse deducted.
9
Payment Processing — <<include>> payment flow triggered; owner completes transaction.
10
Auction Concludes → Winner Announced — Final standings, reports generated, all rosters finalized.

9UI Screens — Admin & Manager Dashboards

The platform provides dedicated dashboards for each actor type. The Admin Dashboard offers a bird's-eye view of all platform operations, while the Manager Dashboard focuses on live auction control and real-time bidding analytics.

9.1 Admin Dashboard

The Admin Dashboard displays comprehensive platform metrics including total auctions (12), total teams (48), with percentage growth indicators. It features an Auction Overview line chart, live auctions list (e.g., "Royal Strikers T20 League Auction, 25 May 2025"), recent auctions table, and a "Top Players by Price" ranking — enabling administrators to monitor platform health at a glance.

Figure 3 — Admin Dashboard: Platform-wide analytics and monitoring
9.2 Manager Dashboard

The Manager Dashboard is the command center during live auctions. It shows the current auction ("Royal Strikers") at 60% progress, with 320 total players (128 sold, 192 remaining) and a total bid value of ₹80,000,000. The right panel displays the live player being auctioned (Jasprit Bumrah, Bowler, India) with current bid ₹520,000 (base ₹200,000), a 28-second countdown timer, and real-time bid history from competing teams. Additional panels show recent activity and team bid rankings.

Figure 4 — Manager Dashboard: Live auction control with real-time bidding
10UI Screens — Auction Pages

Auction pages form the core user interface for both managers and team owners. They provide auction discovery, detailed event views, player pool browsing, team listings, and timeline tracking.

10.1 Upcoming Auctions (Discovery)

The landing page displays all upcoming auctions in a card grid. Each card shows the league name (e.g., "Royal Strikers"), date, base purse, number of joined teams, and a "View Details" button. The header includes search and filter functionality with the tagline "Bid. Build. Win your team."

Figure 5 — Upcoming Auctions: Discovery and filtering page
10.2 Auction Overview Page

Clicking "View Details" opens the full auction page for "Royal Strikers T20 League Auction" (25 May 2025, Mumbai). It shows 12/16 teams joined, ₹80,000,000 prize pool, 3-minute bid timer, and 1 retry allowed. The page includes an Auction Timeline, Top Players in Pool section, an Auction Summary panel, and a "Join Auction" CTA button.

Figure 6 — Auction Overview: Event details, timeline, and top players
10.3 Player Pool

The Player Pool tab lists all available players with their role (Batsman/Bowler), nationality, match statistics, and base price. The currently auctioned player (Jasprit Bumrah) is highlighted with a countdown timer (00:28). Each player row has "Bid Now" and "Watchlist" action buttons. A "Load More Players" pagination is provided at the bottom.

Figure 7 — Player Pool: Browsable player list with bid and watchlist actions
10.4 Team List

The Team List tab displays all teams registered for the auction. Each row shows team name, manager, status (Joined/Open), purse remaining (e.g., ₹40,00,000), player count (e.g., 14/20), and action buttons ("View Team" for joined teams, "Join Now" for open slots).

Figure 8 — Team List: Registered teams with purse and squad status
10.5 Auction Timeline

The Timeline tab visualizes all auction phases with dates and times: Registration Open (10 May), Team Join Deadline, Auction Start (25 May, 3:00 PM), Auction End, and Winner Announcement. A progress bar shows 60% completion with "Live Auction in Progress" status indicator. An "Auction Guide" link provides help documentation.

Figure 9 — Auction Timeline: Phase tracking with progress indicator
