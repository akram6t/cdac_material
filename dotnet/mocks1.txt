Answer Key & Short Explanations
1. c – ArrayList is in System.Collections (non-generic).
2. b – Multicast delegates support chaining with +=.
3. b – Interned string shares reference; compile-time concat may create new.
4. b – Readonly fields are set at declaration or constructor only.
5. a – new hides the method (non-virtual call via base reference).
6. a – sealed prevents inheritance.
7. b – Value type → object (heap).
8. b – Structs are value types; passed by value (copy).
9. b – Designed for mutable string operations.
10. b – Compiler infers type.
11. d – friend is C++/CLI, not C#.
12. b – Local variable shadows static.
13. c – Part of System.Collections.Concurrent.
14. b – Implements IDisposable pattern automatically.
15. b – Classic NullReferenceException.
16. a – Filters >3 → 4,5,9 → descending 9,5,4 → Take(2).
17. c – Multiple constraints are allowed.
18. b – Ensures latest value is read in multi-threaded scenarios.
19. b – Static class + this keyword.
20. b – Async/await resumes after await (non-blocking).
21. d – StartNew is more configurable.
22. b – System.Text.RegularExpressions.
23. a – Stack-allocated or sliced memory without heap alloc.
24. b – Generational GC is core to .NET.
25. a – Indexers use this[].
26. b – Configure() sets up middleware pipeline.
27. b – Scoped (per request) is common for DbContext.
28. b – Model binding from JSON body.
29. d – UseRouting + UseEndpoints.
30. b – Bypasses HTML encoding.
31. c – PATCH for partial updates.
32. b – Global imports.
33. b – [HttpPost].
34. b – ControllerBase for APIs (lighter).
35. b – appsettings.json + environment variants.
36. b – Forward-only, performant reader.
37. a – Primary key.
38. b – Standard migration command.
39. b – Automatic reuse via connection string.
40. b – Needs UseLazyLoadingProxies() or explicit loading.
